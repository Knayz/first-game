# first-game
My first game ever. It has been written on jQuery 3.

## Installation

1. install dependencies

```
    npm i
```

1. install webpack 
```
    sudo npm i -g webpack   
```

   or  
    
```bash
    sudo yarn add -g webpack
```
2. use webpack

```
    webpack
```

3. install node-static

```
    sudo npm i -g node-static
```

4. use node-static

```
    static
```
